# Безопасный Мессенджер 🔐

Простой мессенджер с end-to-end шифрованием сообщений.

## Особенности

- **E2E Шифрование**: Сообщения шифруются на стороне отправителя и расшифровываются только получателем
- **AES-256**: Используется современный алгоритм шифрования
- **Обмен ключами**: Автоматический обмен публичными ключами между пользователями
- **Real-time**: Мгновенная доставка сообщений через WebSocket
- **Красивый UI**: Современный и интуитивно понятный интерфейс

## Установка

```bash
cd messenger
npm install
```

## Запуск

### Сервер:
```bash
npm start
```

### Клиент:
Откройте браузер и перейдите по адресу `http://localhost:3000`

Для тестирования откройте несколько вкладок браузера или разных браузеров.

## Как это работает

1. При входе пользователь генерирует пару ключей (приватный и публичный)
2. Публичный ключ отправляется на сервер и рассылается другим пользователям
3. При отправке сообщения оно шифруется публичным ключом получателя
4. Получатель расшифровывает сообщение своим приватным ключом
5. Сервер видит только зашифрованные данные

## Технологии

- **Node.js** - серверная платформа
- **Socket.IO** - real-time коммуникация
- **CryptoJS** - криптографические операции
- **Express** - веб-сервер
- **HTML/CSS/JS** - клиентский интерфейс

## Безопасность

⚠️ **Важно**: Это демонстрационный проект. Для продакшена рекомендуется:
- Использовать настоящее асимметричное шифрование (RSA, ECC)
- Реализовать протокол Diffie-Hellman для обмена ключами
- Добавить проверку подлинности ключей
- Использовать HTTPS/TLS для транспорта
- Реализовать хранение истории сообщений
